## Modelo de site para Vercel

Foi adicionado um modelo pronto em `vercel_catalogo/` inspirado em página de catálogo de concursos, com:

- `index.html` (estrutura da landing page)
- `styles.css` (layout responsivo)
- `script.js` (cards dinâmicos de módulos/depoimentos/planos)
- `vercel.json` (configuração de deploy)

### Como publicar na Vercel

1. Suba este repositório no GitHub.
2. Na Vercel, clique em **Add New Project**.
3. Importe o repositório e configure o **Root Directory** como `vercel_catalogo`.
4. Deploy.
